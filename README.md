# PNEUMOLOGIA - 5º PERÍODO — Site MedCid

Site offline em HTML/CSS/JS para revisão estratégica de Pneumologia UEA/ESA.

## Como abrir localmente
1. Extraia o ZIP.
2. Abra `index.html` no navegador.
3. Use as abas superiores para navegar.

## Como hospedar no GitHub Pages
1. Crie um repositório no GitHub.
2. Envie `index.html`, `style.css`, `script.js` e a pasta `assets/`.
3. Vá em Settings → Pages → Deploy from branch → main/root.
4. Aguarde o link ser gerado.

## Funcionalidades
- Abas: Início, AP1, AP2, PF, Questões, Cronograma, Mapas e Revisão Final.
- Modo escuro/claro com preferência salva no navegador.
- Banco de questões com filtros por assunto/tipo.
- Gabarito comentado e desempenho salvo localmente.
- Cronograma editável e responsivo.

## Observação
As questões foram transcritas/adaptadas dos materiais anexados e do histórico de provas legível. Em documentos escaneados incompletos, o site preserva o conceito cobrado e evita inventar assuntos fora do material.
